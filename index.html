DRE - TAG

Technologies Used

ReactJS | NodeJS | MongoDB

TIMEFRAME

June 2017 - Oct 201

PROJECT DESCRIPTION

DRE stands for "Damage Recognition Engine". TAG is the term used for Manual purpose.
This tool is made to collect data from damaged car images manually so as to train the ENGINE regarding that for automatic damage/part recognition. One has to manually draw the rectangles on the image regarding the damage whose relevant coordinates gets stored in the database.

Following Technologies were used for the development of DRE - TAG
  • ReactJS (frontend)
  • NodeJS (backend api)
  • MongoDB (database)

Note : It's complete full stack development was done from scratch by myself only
